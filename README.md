# NYC Blog
A blog about NYC. <br />

## Features
* HTML
* Semantic HTML
* `<video>`
* `<embed>`
* `<audio>`
* `controls` attribute for `<video>` and `<audio>` elements
* CSS
* CSS Reset
  * compressed set of CSS rules that *resets* the styling of all HTML elements to a consistent baseline.
